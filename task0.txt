1. Встановіть R та python3. Для windows можна скористатись пакетом Anaconda. Для Linux/MacOS переконайтесь, що встановлені пакети:
   numpy, matplotlib, pandas, scikit-learn, statsmodels.

2. Згенеруйте дві вибірки з 2000 елементів кожна, X - з нормальним розподілом з параметрами 0,1, та Z з розподілом Коші (Cauchy). Скористайтеся data.frame в R
та pandas у python для збереження даних. Для кожної вибірки знайдіть кількість
елементів, що виходить за межі інтервалу [-1,1] та [-3,3]. Знайдіть індекси цих елементів та запишіть їх в окремі вектори (масиви).

3. Намалюйте гістограми для кожної з вибірок, а також їх емпіричні функції розподілу. 

4. Для кожної з вибірок намалюйте графік на якому будуть зображені точки, що попадають всередину інтервалу [-1,1] та зафарбуйте їх зеленим кольором, що попадають в
інтервали [-3,-1) та (1,3] та зафарбуйте їх синім, а також точки, що виходять за межі інтервалу [-3,3] та зафарбуйте їх червоним. По осі x відкладіть номер елемнта, а по осі
y його значення. Розфарбуйте смуги ( [-1,1], [-3,-1) та (1,3], та все що поза [-3,3]) у блідо зелений, синій та червоний кольори.

5. Знайдіть вибіркові середні та дисперсії для обох вибірок, та позначте середнє значення жирною зеленою лінією на малюнках. Оцініть ймовірність того що випадкова величина
з нормальним розподілом (розподілом Коші) відхиляється від середнього більше ніж на три сігма. 

6. Анімуйте Ваші зображення перетворивши їх на відео. Для цього згенеруйте 500 зображень для кожної вибірки, додаючи по чотири нових елементи на кожному зображенні (на першому зображенні має бути лише перші чотири елементи, на другому перших вісім і так далі). Пронумеруйте зображення відповідним чином, зберіжіть у вигляді png файлів, та
скористайтеся утилітою ffmpeg для створення відео (https://stackoverflow.com/questions/24961127/how-to-create-a-video-from-images-with-ffmpeg).
